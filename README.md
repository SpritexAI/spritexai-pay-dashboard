# SpritEXAI Pay — Dashboard

Premium merchant dashboard for [SpritEXAI Pay](https://github.com/SpritexAI/spritexai-pay-core),
built with React + Vite + Tailwind CSS. Charges, ledger reconciliation, paired
devices, and gateway configuration.

Built by **Mohammad Sijan** ([SpritexAI](https://github.com/SpritexAI)).

> Status: scaffolding. Phase 1 target — a clean, restrained, production-grade UI
> over the core engine's v1 API.
